What is an IDE?

IDE (Integrated Development Environment) is used to edit AND compile the code.

Write C++ Using Falcon C++

C++ popular IDE's include :

  • Code::Blocks
  • Eclipse
  • Falcon C++
  • Visual Studio

We are going to use Falcon C++


After the installation process is done, lets write a simple C++ program

Step 1: Open Falcon C++

Ensure that you have installed Falcon C++ in your PC.

If you are having a problem in the installation WATCH HOW TO INSTALL Falcon C++.

Step 2: Write C++ Sample

Write or copy the sample below .

A C++ program :

//simple c++ program to output "Hello codeWJ"
#include <iostream>

using namespace std;

int main()
{
cout<<"Hello, codeWJ";
return 0;
}

Step 3: Save the program

Select File > Save As in the menu.

Name the page as "myfirstprogram.cpp" and click on save .

Step 4: Run the program

Press F9 ro run the program.

Note!! Not all computers support the F9 key to run programs in Falcon C++.
