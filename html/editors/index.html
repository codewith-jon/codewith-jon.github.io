Write HTML Using Visual Studio Code

HTML can be edited by using various HTML editors like:

  • Notepad
  • Atom
  • Sublime Text
  • Visual Studio Code or (VS Code)

For learning HTML we recommend a text editor like Notepad

Follow the steps below to create your first web page with Visual Studio Code.

Step 1: Open VS Code

Ensure that you have installed VS Code in your PC.

If you haven't, WATCH HOW TO INSTALL VS CODE.

Step 2: Write HTML Sample

Write or copy the sample below .

A HTML document:

<!DOCTYPE html >
<html>
<head>
<title>Page Title</title >
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Step 3: Save the HTML doc

Select File > Save As in VS Code menu.

Name the page as "index.html" and click on save .

Step 4: View the HTML doc

In this case, you will use your favorite browse...in my case i'm using chrome
